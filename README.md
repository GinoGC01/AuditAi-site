# 🌐 AuditAi - Sitio Web Oficial

Este repositorio contiene el **sitio web oficial de AuditAi**, una plataforma que utiliza **inteligencia artificial para automatizar procesos contables**.  

AuditAi está diseñado para contadores, estudios contables y empresas que buscan:  
- 📄 **Lectura automática de PDFs y Excel**  
- 💰 **Detección de impuestos, ingresos y gastos**  
- ⚡ **Identificación de inconsistencias y errores**  
- 📊 **Reportes claros y listos para presentar**  
- ⏱️ **Ahorro de tiempo en tareas repetitivas**  

👉 Con AuditAi, reducís errores humanos, optimizás tu tiempo y mejorás la experiencia de tus clientes.  

---

## 📂 Contenido del sitio web

El sitio web incluye varias secciones clave:  

- **Inicio** → Presentación de AuditAi y sus beneficios.  
- **Funciones** → Explicación de cada característica, con ejemplos prácticos.  
- **Nosotros** → Nuestra visión y misión como equipo.  
- **Demo** → Video explicativo mostrando AuditAi en acción.  
- **Contacto** → Formulario para que contadores y empresas puedan comunicarse.  

---

## 🛠️ Tecnologías

El sitio está construido con:  
- [Astro](https://astro.build/) ⚡ Framework moderno para sitios rápidos.  
- [TailwindCSS](https://tailwindcss.com/) 🎨 Estilos y diseño responsivo.  
- [TypeScript](https://www.typescriptlang.org/) ✅ Tipado seguro.
